# Bot Telegram - Devis par IA

## Resume
Assistant Telegram capable de transformer une conversation en devis structure, avec extraction des informations client et generation de lignes de devis.

## Ce que la page doit montrer
- Le besoin metier: generer rapidement des devis sans formulaire long
- L'apport IA: comprendre le langage naturel
- Le resultat: un devis exploitable en base de donnees

## Stack
- Python
- Telegram Bot API
- Mistral API
- SQL
