# Application Mobile Associative

## Resume
Application mobile iOS/Android orientee gestion associative: attribution de taches, messagerie temps reel et partage de fichiers.

## Ce que la page doit montrer
- Le probleme: la coordination complexe dans une association
- La solution: une app unique pour organiser les membres
- La valeur: gain de temps, meilleure communication, suivi clair

## Stack
- React Native
- Node.js / Fastify
- PostgreSQL
- Redis
- Socket.IO
